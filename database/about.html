What is this?

Claude Code can be extended with skills, MCP servers, and connectors -- but they're scattered across dozens of GitHub repos, marketplaces, and registries. This project indexes 346 VC-relevant skills in a searchable dashboard and ships 84 ready-to-use methodology skills covering financial modeling, due diligence, market research, and more.

The methodology skills are self-contained frameworks that work without any API keys or external services. Copy one into your Claude Code skills directory and start using it immediately.

All credit for the skills, frameworks, and implementations belongs to the original authors of the source repositories. See Sources & Attribution for full details.

Contributing

To suggest a skill or report an issue, open an issue on GitHub. Pull requests with new skills are welcome -- add an entry to vc_skills_database.json and run python3 build.py.
