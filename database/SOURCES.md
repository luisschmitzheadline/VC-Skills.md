## Update Instructions

### To refresh the database:

1. **Re-crawl sources** - Visit each URL and compare against current entries
2. **Check for new skills** - Look for skills not in the database
3. **Verify existing skills** - Confirm sources still exist
4. **Update timestamps** - Update crawl dates in this file
5. **Regenerate exports** - Run the regeneration scripts below

### Regeneration Scripts

```bash
python3 build.py
```

This regenerates CSV, dashboard, and workflow docs from `vc_skills_database.json`.
